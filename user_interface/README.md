# Avalon 代码对战平台

Avalon是一个基于Gradio构建的代码对战平台，允许用户编写、测试和比较不同的代码策略。

## 功能特点

- **用户管理**：注册、登录和个人信息管理
- **代码管理**：创建、编辑和调试代码
- **对战管理**：发起测试对战、参与天梯对战
- **天梯排名**：查看总榜和分区榜

## 安装与运行

1. 安装依赖：
   ```console
   pip install pandas
   pip install gradio
   ```

2. 运行应用：（在此目录下）
   ```console
   python app.py
   ```

3. 在浏览器中访问提供的链接（通常是 http://127.0.0.1:7860）

## 使用指南

1. **用户管理**
   - 注册一个新账户
   - 登录您的账户
   - 查看您的个人信息

2. **代码管理**
   - 创建新代码：编写一个包含`play_game()`函数的Python代码，该函数应返回'rock'、'paper'或'scissors'
   - 编辑现有代码
   - 调试您的代码

3. **对战管理**
   - 测试对战：选择您的代码与预设的对手代码进行对战
   - 天梯对战：加入对战队列，与其他用户进行对战
   - 查看对战记录

4. **天梯排名**
   - 查看总榜排名
   - 查看分区榜排名

## 游戏规则

- 目前实现了简单的石头剪刀布游戏
- 您的代码需要定义一个`play_game()`函数，返回'rock'、'paper'或'scissors'
- 胜负规则：石头胜剪刀，剪刀胜布，布胜石头
